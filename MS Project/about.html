This application is part of a master project proposed by Prof. Martin Gander & Prof. Pierre-Alain Cherix from the Mathematics Department of University of Geneva. The aim of this project is to free players from using predefined data embedded by game designer. We introduce a knowledge importer which allows players to enter their own data and then the game interface will use these user-defined data in the game.
Shuo YAN 2015, supervised by Prof. Martin Gander & Prof. Pierre-Alain Cherix
